Exercício: Criando uma Interface de Veículos
Objetivo: Praticar a criação e implementação de interfaces em Java.

Descrição: Crie uma interface chamada Veiculo que tenha os seguintes métodos:
void acelerar();
void frear();
void abastecer(int litros);

Crie duas classes que implementem essa interface:
Carro
Moto

Cada classe deve implementar os métodos da interface de forma adequada.
Adicionalmente, adicione uma variável para representar a quantidade de combustível e atualize ela no método abastecer.

Regras:
Quando acelerar for chamado, imprima uma mensagem como: "O carro acelerou!" ou "A moto acelerou!".
Quando frear for chamado, imprima uma mensagem como: "O carro freou!" ou "A moto freou!".
No método abastecer, aumente a quantidade de combustível e imprima o novo total.

Extras (opcional):
No main(), instancie objetos de Carro e Moto e chame os métodos para demonstrar seu funcionamento.
